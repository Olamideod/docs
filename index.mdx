## Install the SDK

<CodeGroup>

```bash npm
npm install @protekt/javascript
```

```bash pnpm
pnpm add @protekt/javascript
```

```bash yarn
yarn add @protekt/javascript
```

```bash python
pip install protekt
```

```bash go
go get github.com/protekt/go
```

</CodeGroup>

## Get started in 3 steps

<Steps>
  <Step title="Create your project">
    Sign up, create an app in the dashboard, and grab your client ID.
  </Step>
  <Step title="Initialize the SDK">
    Add the SDK to your app and set the redirect URL.
  </Step>
  <Step title="Protect a route">
    Verify sessions on the server or guard client routes and you’re live.
  </Step>
</Steps>

## Choose your stack

<Tabs>
  <Tab title="JavaScript">

```javascript
import Protekt from '@protekt/javascript';
const protekt = new Protekt({ clientId: 'pk_live_xxx', redirectUri: '/callback' });
await protekt.init();
```

  </Tab>
  <Tab title="Python">

```python
from protekt import Protekt
protekt = Protekt(client_id='pk_live_xxx')
```

  </Tab>

## Frequently asked

<AccordionGroup>
  <Accordion title="Is Protekt enterprise ready?">
    Yes. SSO (SAML/OIDC), MFA, RBAC, audit logs, SCIM, and fine‑grained API permissions are built in.
  </Accordion>
  <Accordion title="How do I validate tokens?">
    Use the server SDK middleware or call the `validate` endpoint. See the API Reference for examples.
  </Accordion>
</AccordionGroup>
