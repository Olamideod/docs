# 5-Minute Quickstart

You're about to add secure login to your app in under 5 minutes. Let's get started.

## Prerequisites

- Node.js 18+ installed
- A code editor
- 5 minutes of your time

## Step 1: Create Your Protekt Project

Sign up at [protekt.dev](https://protekt.dev) and create a new project.

Copy your **Client ID** and **Client Secret** from the dashboard.

## Step 4: Configure Environment

Create a `.env` file in your project root:

```bash  .env
PROTEKT_CLIENT_ID=your_client_id_here
PROTEKT_CLIENT_SECRET=your_client_secret_here
PROTEKT_REDIRECT_URI=http://localhost:3000/callback
```

<Info>Never commit secrets to version control. Keep them in environment variables.</Info>

## Step 5: Add Authentication

<Tabs>
<Tab title="React + Express">

**Frontend Setup**
```jsx  App.jsx
import { ProtektProvider, useProtekt } from '@protekt/react';

function LoginButton() {
  const { user, login, logout, isLoading } = useProtekt();
  
  if (isLoading) return <div>Loading...</div>;
  if (!user) return <button onClick={login}>Sign In</button>;
  
  return (
    <div>
      <h1>Welcome, {user.name}!</h1>
      <button onClick={logout}>Sign Out</button>
    </div>
  );
}

export default function App() {
  return (
    <ProtektProvider clientId={process.env.REACT_APP_PROTEKT_CLIENT_ID}>
      <LoginButton />
    </ProtektProvider>
  );
}
```

**Backend Setup**
```javascript  server.js
import express from 'express';
import { Protekt } from '@protekt/node';

const app = express();
const protekt = new Protekt({
  clientId: process.env.PROTEKT_CLIENT_ID,
  clientSecret: process.env.PROTEKT_CLIENT_SECRET
});

// Handle auth callback
app.get('/callback', async (req, res) => {
  const { user } = await protekt.handleCallback(req.query.code);
  req.session.user = user;
  res.redirect('/dashboard');
});

// Protect routes
app.get('/api/profile', protekt.requireAuth, (req, res) => {
  res.json({ user: req.user });
});

app.listen(3001);
```

```jsx  pages/_app.js
import { ProtektProvider } from '@protekt/nextjs';

export default function App({ Component, pageProps }) {
```

**Add Login**
```jsx  pages/index.js
import { useProtekt } from '@protekt/nextjs';

export default function Home() {

    <div>

}
```

**API Route**
```javascript  pages/api/profile.js
import { protekt } from '@protekt/nextjs';

export default protekt.withAuth(async (req, res) => {
  res.json({ user: req.user });
});
```

</Tab>
<Tab title="Vue + FastAPI">

**Frontend Setup**
```vue  App.vue
<template>
  <div>
    <div v-if="isLoading">Loading...</div>
    <button v-else-if="!user" @click="login">Sign In</button>
    <div v-else>
      <h1>Welcome, {{ user.name }}!</h1>
      <button @click="logout">Sign Out</button>
    </div>
  </div>
</template>

<script setup>
import { useProtekt } from '@protekt/vue';

const { user, login, logout, isLoading } = useProtekt({
  clientId: import.meta.env.VITE_PROTEKT_CLIENT_ID
});
</script>
```

**Backend Setup**
```python  main.py
from fastapi import FastAPI, Depends
from protekt import Protekt, get_current_user

app = FastAPI()
protekt = Protekt(
    client_id="your_client_id",
    client_secret="your_client_secret"
)

@app.get("/callback")
async def callback(code: str):
    user = await protekt.handle_callback(code)
    return {"user": user}

@app.get("/api/profile")
async def profile(user = Depends(get_current_user)):
    return {"user": user}
```
