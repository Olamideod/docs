## Manual Setup

If you prefer to set up manually, follow these steps:

### 1. Create Your Account

1. Visit [dashboard.protekt.com](https://dashboard.protekt.com)
2. Sign up for a free account
3. Create your first application

### 2. Configure Your Application

```javascript
// Initialize Protekt SDK
import { Protekt } from '@protekt/sdk';

const protekt = new Protekt({
  domain: 'your-tenant.protekt.com',
  clientId: 'your-client-id',
  redirectUri: 'https://your-app.com/callback'
});
```

### 3. Add Authentication

```javascript
// Redirect to login
await protekt.login();

// Handle callback
protekt.handleCallback();
```

### 4. Protect Your Routes

```javascript
// Check authentication status
if (await protekt.isAuthenticated()) {
  // User is logged in
  const user = await protekt.getUser();
} else {
  // Redirect to login
  await protekt.login();
}
```
