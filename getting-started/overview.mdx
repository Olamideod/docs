# What You Can Build with Protekt

You're five minutes away from production-ready auth. Build everything from simple login screens to enterprise SSO with one platform.

## What Protekt Enables

**SaaS Login Systems** - Add user registration, login, and password reset in minutes.

**Enterprise SSO Integration** - Connect Google Workspace, Microsoft, Okta, and any SAML provider.

**Multi-Factor Authentication** - Add TOTP, SMS, and biometric auth with built-in recovery options.

**B2B Multi-Tenant Auth** - Create organizations and manage team permissions automatically.

## Key Features

**JWT-First Architecture** - Stateless tokens with auto-refresh and custom claims.

**Session Management** - Secure server-side sessions with Redis backing.

**Organization Management** - Built-in multi-tenancy with team invites and role assignments.

**Real-Time Webhooks** - Get notified when users sign up or suspicious logins occur.

**Global Performance** - Sub-200ms token validation worldwide.

## Auth Flows

<Tabs>
<Tab title="Standard Login">

```javascript  server.js
import { Protekt } from '@protekt/node';

// Redirect to login
app.get('/login', (req, res) => {
  const loginUrl = protekt.getLoginUrl({
    redirectUri: 'https://yourapp.com/callback'
  });
  res.redirect(loginUrl);
});

// Handle callback
app.get('/callback', async (req, res) => {
  const { user, tokens } = await protekt.handleCallback(req.query.code);
  req.session.user = user;
  res.redirect('/dashboard');
});
```

</Tab>
<Tab title="Magic Link">

```javascript  auth.js
// Send magic link
app.post('/auth/magic-link', async (req, res) => {
  await protekt.sendMagicLink({
    email: req.body.email,
    redirectTo: 'https://yourapp.com/dashboard'
  });
  res.json({ sent: true });
});
```

</Tab>
<Tab title="SSO Integration">

```javascript  sso.js
// Enterprise SSO redirect
app.get('/sso/:domain', async (req, res) => {
  const ssoUrl = await protekt.getSSOUrl({
    domain: req.params.domain,
    redirectUri: 'https://yourapp.com/sso/callback'
  });
  res.redirect(ssoUrl);
});
```

## Teams & Organizations

Handle complex B2B auth patterns with zero setup:

```javascript  organizations.js
// Create organization
const org = await protekt.organizations.create({
  name: 'Acme Corp',
  domain: 'acme.com',
  settings: { requireMFA: true }
});

// Invite team members
await protekt.organizations.inviteUser({
  organizationId: org.id,
  email: 'john@acme.com',
  role: 'admin'
});

// Check permissions
const hasAccess = await protekt.checkPermission(
  req.user.id, 
  req.params.orgId, 
  'users:delete'
);
```

## Roles & Permissions

Build granular access control in minutes:

```javascript  rbac.js
// Define custom roles
const roles = {
  admin: ['users:*', 'billing:*', 'settings:*'],
  manager: ['users:read', 'users:invite'],
  member: ['users:read']
};

// Protect API routes
app.use('/api/admin', protekt.requirePermission('admin:access'));
```

## Secure Sessions

Choose JWT tokens or server-side sessions:

<Tabs>
<Tab title="JWT Tokens">

```javascript  jwt-auth.js
// Validate JWT middleware
app.use('/api', protekt.validateJWT);

// Access user from token
app.get('/api/profile', (req, res) => {
  res.json({
    user: req.user,
    permissions: req.user.permissions,
    organization: req.user.organization
  });
});
```

</Tab>
<Tab title="Server Sessions">

```javascript  sessions.js
// Session middleware
app.use(protekt.sessionMiddleware);

// Revoke all sessions
app.post('/api/logout-everywhere', async (req, res) => {
  await protekt.revokeAllSessions(req.user.id);
  res.json({ success: true });
});
```

</Tab>
</Tabs>


**How It Works:**
1. Your app redirects to Protekt for login
2. Users authenticate with their preferred method  
3. Protekt returns secure tokens to your app
4. You validate tokens and access user data
